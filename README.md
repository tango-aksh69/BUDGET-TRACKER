# Budget Tracker

A simple and responsive budget tracking website built using HTML, CSS, and JavaScript.  
It helps users plan and manage their expenses by creating and monitoring budgets for various categories.

---

## Features
- Create and manage multiple budget categories.  
- Add or remove expenses and view remaining balance.  
- Real-time budget updates.  
- Includes login, registration, and about pages.  
- Clean and user-friendly interface.

---

## How to Use
1. Login or register to start using the tracker.  
2. Add your budget categories such as food, travel, or entertainment.  
3. Set a budget and record expenses as they occur.  
4. View the updated balance and manage your spending efficiently.
